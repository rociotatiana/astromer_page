Tutorials


Install ASTROMER

ASTROMER is tested and supported on the following 64-bit systems:

  • Python 3.8-3.10

    • Install our python library including all the functionalities:

    pip install ASTROMER


    Explore ASTROMER

    The following tutorials were made for you to make a better understanding of the nature of it.


    Tutorial 1: Prepare your data

    Either numpy or tf.records, here you will learn how to format your own data to fit ASTROMER's requirements.



    Tutorial 2: Classification using MACHO Catalog

    Use the pre-trained weights to classify light curves from the Massive Compact Halo Object survey.



    Tutorial 3: Classification using ALeRCE broker

    Use the pre-trained weights to classify light curves from the Zwicky Transient Facility (ZTF).
